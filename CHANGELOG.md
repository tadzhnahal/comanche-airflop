# Changelog

## 0.1.0 - 2026-06-24

### Added

- Добавлен интерактивный canvas для карты зависимостей.
- Добавлены компоненты и зависимости.
- Добавлены сценарии создания, редактирования и удаления компонентов.
- Добавлены сценарии создания, редактирования и удаления зависимостей.
- Добавлен анализ downstream impact по выбранному компоненту.
- Добавлены handle-точки на узлах графа.
- Добавлено создание связей через handle-точки.
- Добавлен перенос концов связей между узлами и сторонами узлов.
- Добавлено хранение source_handle и target_handle в backend.
- Добавлен Streamlit UI для работы с graph canvas.
- Добавлен FastAPI backend.
- Добавлена PostgreSQL-база для компонентов и зависимостей.
- Добавлен Docker Compose для локального запуска проекта.

### Changed

- Улучшен внешний вид canvas.
- Улучшено поведение pan и zoom.
- Улучшено поведение выбора узлов и связей.
- Убраны лишние Streamlit-блоки вокруг canvas.
